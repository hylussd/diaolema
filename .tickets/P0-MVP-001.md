## 目标

交付可运行的微信小程序MVP，验证"地图+标点+天气+禁钓区"基础工具价值。

## P0 功能清单（已完成）

1. ✅ 地图展示（卫星/路网双图层，精准GPS定位）
2. ✅ 个人标点CRUD（上传/编辑/删除，补充地形/鱼种/备注）
3. ✅ 标点收藏分类（分类文件夹，批量管理）
4. ✅ 基础天气展示（温度/气压/风力/湿度，和风API）
5. ✅ 禁钓区标注（红色半透明多边形，显示原因/时间）
6. ✅ 禁钓区预警（用户进入弹窗警告）
7. ✅ 微信分享（生成链接分享给好友）

## 启动方式

**后端：**
```bash
cd 05-开发/server
uv pip install -r requirements.txt
uv run python -m app.init_db   # 初始化数据库
uv run python -m app.main     # 启动服务
```

**前端：**
1. 微信开发者工具导入 `05-开发/miniapp/`
2. 填入真实 appid（project.config.json）
3. 勾选「不校验合法域名」连接本地后端
4. 放入 TabBar 图标（assets/icons/，8张81×81px PNG）
