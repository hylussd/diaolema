# 「钓了吗」钓鱼标点地图完整产品方案（按功能优先级）

### 一、产品定位与核心价值

#### 定位

以 "多维度数据融合 + AI 智能标点推荐" 为核心竞争力，打造 "工具 + 社区 + 商城" 一体化钓鱼垂直平台，为钓友提供 "精准找标点、安全作钓、高效交流、便捷购物" 的全链路解决方案。

#### 核心价值

- 核心差异：打破传统 "盲目找标点" 模式，通过数据融合与 AI 算法，实现 "科学选点、精准作钓"
- 工具价值：满足标点管理、禁钓区规避、鱼情查询等刚需，降低作钓门槛
- 生态价值：通过社区沉淀优质内容，以场景化商城实现商业闭环，提升用户粘性与变现能力

### 二、核心功能完整设计（按优先级排序）

#### 一、一级核心（核心卖点，产品竞争力核心）

##### 1. 多维度数据融合查询 + AI 智能标点推荐

- 数据融合体系（全维度覆盖）
  - 地图数据：卫星 / 路网双图层、精准定位（支持 GPS / 北斗）、离线地图下载（按区域缓存，节省流量）
  - 天气数据：实时温湿度、气压、风力风向、7 天趋势曲线、极端天气（暴雨 / 大风 / 高温）弹窗预警
  - 水文数据：
    - 内陆水域：水温、流速、溶氧量、水质透明度（众包 + 官方数据）
  - 天文数据：月相（新月 / 满月等相位标注）、日出日落时间、Solunar 周期（鱼群活跃时段预测）

- AI 智能推荐核心能力（差异化亮点）
  - 自动推荐模式：
    - 用户输入目标条件（日期 / 时段、钓场类型、目标鱼种），系统自动融合实时 / 预测数据，AI 推荐最优标点（地图高亮显示）
    - 推荐理由可视化：标注 "推荐依据"（如 "气压 1005hPa + 水温 24℃+ 浅滩地形，适配鲫鱼活性高峰"）
  - 自定义参数配置：
    - 可调整参数清单：水温区间（5-35℃）、气压阈值（950-1050hPa）、溶氧量范围（5-12mg/L）、目标鱼种（鲫鱼 / 鲈鱼等）、钓法偏好（台钓 / 路亚等）、地形偏好（浅滩 / 深潭等）
    - 配方管理：支持保存自定义参数组合为 "个人配方"（命名、备注），可复用、分享给微信好友
  - 高级筛选功能：
    - 单一参数筛选：如 "仅显示气压 990-1010hPa 的标点"
    - 多参数组合筛选：如 "水温 20-25℃ + 鲤鱼 + 水库钓场 + 溶氧量≥6mg/L"
    - 快速筛选标签：提供 "今日最佳""新手友好""大物标点" 等预设标签
  - 数据可视化呈现：
    - 趋势图表：温湿度 / 气压 / 溶氧量 7 天变化曲线，标注鱼情活跃区间
    - 数据卡片：每个标点关联核心数据摘要（如 "当前气压 1008hPa | 水温 23℃ | 鲫鱼活跃度 85%"）

#### 二、二级核心（基础工具刚需，支撑核心卖点落地）

##### 2. 标点全生命周期管理（收藏 + 分享 + 上传）

- 个人标点管理：
  - 上传功能：支持手动标注位置，补充地形（浅滩 / 深潭等）、鱼种、作钓备注，默认 "仅个人可见"
  - 收藏分类：可创建 "自留标点""新手练习" 等分类文件夹，支持批量移动、导出（KML 格式）
  - 编辑与删除：随时修改标点信息，支持逻辑删除（回收站保留 30 天）
- 分级分享机制（兼顾隐私与共享）：
  - 微信定向分享：生成加密临时链接（有效期 1-7 天可设置），仅接收方可见，禁止二次转发
  - 平台社区分享：主动选择 "分享至社区"，需提交鱼获照片 / 视频作为真实性证明，经 AI（违规检测）+ 人工（真实性审核）通过后，入驻公共标点库（支持匿名发布）
  - 分享激励：优质分享标点（高点赞 / 高使用量）可获得积分，用于兑换商城优惠券 / 会员权益
- 社区标点审核：
  - 审核标准：禁钓区排查、信息真实性（对比卫星图 / 鱼获证明）、无违规内容
  - 驳回反馈：明确告知驳回原因（如 "未提供有效鱼获证明""位于季节性禁钓区"）

##### 3. 禁钓区精准管控（合规底线）

- 数据来源：对接国家农业农村部、地方渔政部门公开数据，实时同步禁钓区信息
- 可视化呈现：
  - 地图标注：红色半透明多边形标注禁钓区，区分 "永久禁钓""季节性禁钓"（标注生效 / 失效时间）
  - 详情展示：点击禁钓区，显示禁钓原因（如 "长江十年禁渔"）、管理部门、相关法规链接
- 合规管控功能：
  - 定位预警：用户定位在禁钓区时，弹窗预警 "当前区域为禁钓区，禁止作钓与标点上传"
  - 上传拦截：标注位置在禁钓区时，直接拦截上传，提示合规风险
  - 搜索屏蔽：筛选标点时自动排除禁钓区点位

##### 4. 钓点地图（鱼情 + 鱼种 + 历史鱼获）

- 核心信息展示：
  - 基础信息：每个标点标注钓场类型（水库 / 野河 / 海钓等）、地形特征、目标鱼种（多鱼种标签）
  - 鱼情数据：当前可钓指数（结合天气 / 水文计算）、鱼种活跃度排名（如 "1. 鲫鱼 85% 2. 鲤鱼 72%"）
  - 历史记录：钓友打卡的历史鱼获（鱼体大小、咬钩时段、钓法），按时间排序展示
- 实时动态更新：
  - 最新鱼获：钓友 24 小时内的打卡记录（鱼获照片 + 文字描述），地图标注 "新鱼获" 标签
  - 鱼情预警：某标点近期鱼获骤减时，提示 "鱼情下滑，建议调整作钓时间 / 钓法"

#### 三、三级核心（生态延展，提升用户粘性与商业变现）

##### 5. 垂直社区互动（精准社交 + 内容沉淀）

- 核心社区板块：
  - 标点关联社区：每个公共标点附带专属讨论区，交流钓法技巧、鱼情反馈、避坑指南
  - 内容分类专区：
    - 教程专区：新手入门、进阶技巧、钓法教学（图文 / 视频）、装备使用教程，支持按鱼种 / 钓法筛选
    - 鱼获分享：关联标点打卡，展示鱼获照片 / 视频 + 作钓数据（天气 / 钓组 / 饵料），支持点赞、评论、收藏
    - 吐槽专区：分享踩坑经历（如 "虚假标点曝光""装备吐槽"）、作钓趣事，释放情绪价值
- 互动功能：
  - 基础互动：点赞、评论、@好友、转发内容
  - 内容标签：支持添加 "台钓""路亚""冬季钓" 等标签，提升检索效率
  - 优质内容推荐：基于用户关注鱼种 / 钓法，智能推荐相关内容

##### 6. 场景化商城（商业闭环）

- 商品分类体系：
  - 一级分类：鱼竿、鱼线、饵料、浮漂、配件、服饰、电子设备（探鱼器 / 打窝船等）
  - 二级分类：如鱼竿→台钓竿 / 路亚竿 / 溪流竿 / 海竿，饵料→腥香型 / 清香型 / 酸臭型 / 窝料
- 场景化推荐逻辑：
  - 核心推荐：基于当前钓点类型（如水库）、目标鱼种（如鲤鱼）、天气数据（如 25℃），推荐适配装备（如 "水库鲤鱼专用饵料""25℃高活性窝料"）
  - 配方关联：用户使用自定义参数配方时，推荐适配的饵料 / 钓组（如 "低气压配方→腥香型饵料"）
  - 热门推荐：钓友打卡高频使用的装备、季节爆款（如 "冬季鲫鱼专用饵料"）
- 核心功能：
  - 商品详情：展示参数、适用场景、用户评价（关联钓点使用反馈）、售后保障
  - 组合套餐：按场景搭配（如 "台钓入门套餐""路亚鲈鱼套餐"），性价比更高
  - 交易功能：支持微信支付、订单跟踪、物流查询、售后申请
  - 积分兑换：社区互动 / 标点分享获得的积分，可兑换商品优惠券、小额商品

### 三、技术支撑体系（适配核心功能落地）

#### 1. 核心技术选型

- 前端：微信原生小程序（WXML/WXSS/JS）+ ECharts（数据可视化）
- 后端：Java + Spring Boot 2.x + Spring Cloud（微服务）
- 数据库：MySQL（核心数据）+ Redis（缓存热点数据）
- 存储：阿里云 OSS（图片 / 视频 / 离线地图存储）
- 算法：AI 推荐模型（基于协同过滤 + 鱼种习性规则库）

#### 2. 第三方服务对接

| 服务类型 | 对接平台 | 核心用途 |
| --- | --- | --- |
| 地图服务 | 微信地图 API | 定位、地图展示、地理围栏判断 |
| 天气数据 | 和风天气 API | 实时 / 趋势天气数据、预警推送 |
| 水文数据 | 国家水文站 API + 众包 | 水温、流速、溶氧量数据 |
| 禁钓区数据 | 国家农业农村部 + 地方渔政 API | 禁钓区边界、规则同步 |
| 内容审核 | 阿里云内容安全 API | 鱼获照片 / 文字违规检测 |
| 支付服务 | 微信支付 API | 商城交易、订单支付 |

### 四、数据安全与合规设计

1. 数据安全：
   - 敏感数据加密：用户定位信息、微信 OpenID 等脱敏存储，传输过程 HTTPS 加密
   - 接口安全：所有接口加签验签，防止篡改；Token 有效期控制（2 小时）
   - 存储安全：数据库每日增量备份 + 每周全量备份，OSS 文件设置访问权限

2. 合规设计：
   - 隐私政策：明确数据收集范围（仅必要信息），支持用户自主删除数据
   - 内容合规：用户上传内容经 AI + 人工审核，杜绝违规信息
   - 垂钓引导：首页展示文明垂钓倡议（钓获放流、垃圾带走），宣传禁渔政策

### 五、商业变现路径（按落地阶段）

1. 初期：商城商品销售（自营 + 第三方入驻佣金）
2. 中期：
   - 付费会员：解锁高级 AI 推荐、优质标点库、免广告权益
   - 积分体系：关联社区互动与商城消费，提升用户活跃度与复购率
3. 长期：
   - 装备品牌广告：精准投放（如路亚装备定向推送给路亚钓友）
   - 线下活动：钓鱼赛事招商、师父带钓付费服务
