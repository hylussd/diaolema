# 钓了吗 — 项目元数据

## 基本信息

- **项目名称：** 钓了吗
- **产品定位：** 数据融合型钓鱼标点地图（内陆淡水优先）
- **产品类型：** 微信小程序
- **创建日期：** 2026-04-14
- **当前阶段：** 方案规划中

## 核心约束

| 维度 | 决策 |
|---|---|
| 数据来源优先级 | 内陆淡水（水库/野河/湖泊） |
| MVP目标用户 | 老钓手（经验优先，不需要新手引导） |
| 技术栈偏好 | Python快速迭代优先 |
| 预计用户量级 | 100人（种子期） |
| 开发人力 | OpenClaw |

## 技术选型

| 层 | 选型 |
|---|---|
| 前端 | 微信小程序（WXML/WXSS/JS） |
| 后端 | Python FastAPI |
| 数据库 | SQLite（早期）→ MySQL |
| 地图 | 微信地图API |
| 天气 | 和风天气API |

## 风险提示

1. **水文数据缺口** — 内陆淡水无官方API，水温/溶氧量/流速依赖用户众包上报
2. **AI推荐质量** — 无真实鱼获数据前，推荐基于规则引擎，不宜夸大效果
3. **社区冷启动** — 需设计众包激励机制补充数据密度
