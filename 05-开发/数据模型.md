# 钓了吗 P0 MVP 数据模型

## 1. 数据库概览

- **引擎：** SQLite + aiosqlite（异步）
- **ORM：** SQLAlchemy 2.0 异步模式
- **文件路径：** `./data/diaolema.db`（服务器端，单文件）
- **迁移策略：** 初期手动管理 schema，后续引入 Alembic

---

## 2. ER 关系图（文字版）

```
users (1) ──── (N) fishing_spots
users (1) ──── (N) categories
users (1) ──── (N) favorites

categories (1) ──── (N) fishing_spots (一个分类含多个标点)

forbidden_zones (独立表，无用户归属)
weather_cache (独立表，按 location_key 缓存)
```

---

## 3. 表结构

### 3.1 users（用户表）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 用户 ID |
| openid | TEXT | UNIQUE, NOT NULL, INDEX | 微信 openid，等同用户身份 |
| nickname | TEXT | | 微信昵称（冗余存储） |
| avatar_url | TEXT | | 微信头像 URL |
| created_at | DATETIME | NOT NULL, DEFAULT NOW | 注册时间 |
| updated_at | DATETIME | NOT NULL | 更新时间 |

> **说明：** 简化版不做登录注册，用户以 openid 标识身份。

---

### 3.2 categories（分类表）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 分类 ID |
| user_id | INTEGER | FK → users.id, NOT NULL, INDEX | 归属用户 |
| name | TEXT | NOT NULL | 分类名称（如「常去」「打算去」） |
| color | TEXT | DEFAULT '#3B82F6' | 地图上该分类标点的颜色 |
| sort_order | INTEGER | DEFAULT 0 | 排序权重 |
| created_at | DATETIME | NOT NULL, DEFAULT NOW | 创建时间 |

> **约束：** UNIQUE(user_id, name)

---

### 3.3 fishing_spots（标点表）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 标点 ID |
| user_id | INTEGER | FK → users.id, NOT NULL, INDEX | 归属用户 |
| category_id | INTEGER | FK → categories.id, NULL, INDEX | 所属分类（可为空） |
| name | TEXT | NOT NULL | 标点名称 |
| latitude | REAL | NOT NULL | 纬度（GCJ-02） |
| longitude | REAL | NOT NULL | 经度（GCJ-02） |
| terrain | TEXT | | 地形：河流/湖泊/水库/池塘/其他 |
| fish_species | TEXT | | 鱼种 JSON 数组：["鲫鱼","鲤鱼","草鱼"] |
| fishing_method | TEXT | | 钓法：台钓/传统钓/路亚/海竿 |
| water_depth | REAL | | 水深（米） |
| water_clarity | TEXT | | 水质：清澈/微浊/浑浊 |
| price_info | TEXT | | 收费信息：免费/收费+金额 |
| description | TEXT | | 备注描述 |
| photos | TEXT | | 图片 URL 列表 JSON |
| is_public | INTEGER | DEFAULT 0 | 是否公开分享（0=私有，1=公开） |
| extra | TEXT | | 扩展字段 JSON，预留 |
| created_at | DATETIME | NOT NULL, DEFAULT NOW | 创建时间 |
| updated_at | DATETIME | NOT NULL | 更新时间 |

> **索引：** (user_id, category_id) 联合索引，地理坐标可加 R-Tree（SQLite 扩展）
> **扩展预留：** `extra` JSON 字段支撑未来字段扩展，不改动表结构

---

### 3.4 forbidden_zones（禁钓区表）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 禁钓区 ID |
| name | TEXT | NOT NULL | 禁钓区名称 |
| polygon | TEXT | NOT NULL | 多边形坐标 JSON：[[lng,lat], [lng,lat], ...] |
| reason | TEXT | | 禁钓原因：生态保护/行洪安全/其他 |
| start_time | DATETIME | | 禁钓开始时间（可为 NULL 表示永久） |
| end_time | DATETIME | | 禁钓结束时间（可为 NULL 表示永久） |
| source | TEXT | | 数据来源：政府公告/用户上报 |
| created_at | DATETIME | NOT NULL, DEFAULT NOW | 创建时间 |
| updated_at | DATETIME | NOT NULL | 更新时间 |

> **说明：** 禁钓区由管理员维护（未来扩展 user.role='admin'），初期可硬编码或从配置文件导入

---

### 3.5 weather_cache（天气缓存表）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 缓存 ID |
| location_key | TEXT | NOT NULL, INDEX | 位置标识（格式：lat,lng 保留4位小数） |
| raw_response | TEXT | | 和风 API 原始返回 JSON |
| now_temp | REAL | | 当前温度（℃） |
| now_pressure | REAL | | 气压（hPa） |
| now_wind_speed | REAL | | 风速（m/s） |
| now_humidity | INTEGER | | 湿度（%） |
| now_weather_text | TEXT | | 天气文字：晴/多云/雨 |
| now_feels_like | REAL | | 体感温度 |
| hourly_forecast | TEXT | | 逐小时预报 JSON |
| cached_at | DATETIME | NOT NULL | 缓存时间 |
| expires_at | DATETIME | NOT NULL | 过期时间（缓存时间 + 5分钟） |

> **TTL：** 5 分钟，避免频繁调用和风 API
> **查询策略：** 先查 `expires_at > NOW()`，命中则返回缓存，未命中则调 API 更新

## 4. SQL 建表语句

```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    openid TEXT NOT NULL UNIQUE,
    nickname TEXT,
    avatar_url TEXT,
    created_at DATETIME NOT NULL DEFAULT (datetime('now')),
    updated_at DATETIME NOT NULL DEFAULT (datetime('now'))
);
CREATE INDEX idx_users_openid ON users(openid);

CREATE TABLE categories (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    name TEXT NOT NULL,
    color TEXT DEFAULT '#3B82F6',
    sort_order INTEGER DEFAULT 0,
    created_at DATETIME NOT NULL DEFAULT (datetime('now')),
    FOREIGN KEY (user_id) REFERENCES users(id),
    UNIQUE(user_id, name)
);
CREATE INDEX idx_categories_user_id ON categories(user_id);

CREATE TABLE fishing_spots (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    category_id INTEGER,
    name TEXT NOT NULL,
    latitude REAL NOT NULL,
    longitude REAL NOT NULL,
    terrain TEXT,
    fish_species TEXT,
    fishing_method TEXT,
    water_depth REAL,
    water_clarity TEXT,
    price_info TEXT,
    description TEXT,
    photos TEXT,
    is_public INTEGER DEFAULT 0,
    extra TEXT,
    created_at DATETIME NOT NULL DEFAULT (datetime('now')),
    updated_at DATETIME NOT NULL DEFAULT (datetime('now')),
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (category_id) REFERENCES categories(id)
);
CREATE INDEX idx_spots_user_id ON fishing_spots(user_id);
CREATE INDEX idx_spots_category_id ON fishing_spots(category_id);
CREATE INDEX idx_spots_user_category ON fishing_spots(user_id, category_id);

CREATE TABLE forbidden_zones (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    polygon TEXT NOT NULL,
    reason TEXT,
    start_time DATETIME,
    end_time DATETIME,
    source TEXT,
    created_at DATETIME NOT NULL DEFAULT (datetime('now')),
    updated_at DATETIME NOT NULL DEFAULT (datetime('now'))
);

CREATE TABLE weather_cache (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    location_key TEXT NOT NULL,
    raw_response TEXT,
    now_temp REAL,
    now_pressure REAL,
    now_wind_speed REAL,
    now_humidity INTEGER,
    now_weather_text TEXT,
    now_feels_like REAL,
    hourly_forecast TEXT,
    cached_at DATETIME NOT NULL DEFAULT (datetime('now')),
    expires_at DATETIME NOT NULL,
    UNIQUE(location_key)
);
CREATE INDEX idx_weather_location_key ON weather_cache(location_key);
CREATE INDEX idx_weather_expires ON weather_cache(expires_at);
```

## 5. 索引策略

| 表 | 索引 | 用途 |
|----|------|------|
| users | openid | 快速用户身份查询 |
| categories | user_id | 查某用户所有分类 |
| fishing_spots | user_id | 查某用户所有标点 |
| fishing_spots | (user_id, category_id) | 按分类查标点 |
| weather_cache | location_key | 天气缓存命中 |
| weather_cache | expires_at | 缓存过期清理 |
