## 2. 数据模型扩展

### 2.1 新增表：spot_checkins（历史打卡记录）

> **说明：** 用户在标点详情页手动记录本次钓鱼结果（鱼种、天气），仅本地记录，不上报不共享。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTOINCREMENT | 记录 ID |
| user_id | INTEGER | FK → users.id, NOT NULL, INDEX | 归属用户 |
| spot_id | INTEGER | FK → fishing_spots.id, NOT NULL, INDEX | 关联标点 |
| fish_caught | TEXT | | 钓到的鱼种 JSON：["鲫鱼","鲤鱼"] |
| weight_kg | REAL | | 渔获总重量（kg） |
| weather_text | TEXT | | 当日天气（冗余存储，减少关联查询） |
| temp | REAL | | 当日温度（℃） |
| pressure | REAL | | 当日气压（hPa） |
| notes | TEXT | | 备注 |
| checkin_time | DATETIME | NOT NULL, DEFAULT NOW | 打卡时间 |
| created_at | DATETIME | NOT NULL, DEFAULT NOW | 创建时间 |

> **索引：** (user_id, spot_id)、checkin_time（用于按时间排序展示）

```sql
CREATE TABLE spot_checkins (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    spot_id INTEGER NOT NULL,
    fish_caught TEXT,
    weight_kg REAL,
    weather_text TEXT,
    temp REAL,
    pressure REAL,
    notes TEXT,
    checkin_time DATETIME NOT NULL DEFAULT (datetime('now')),
    created_at DATETIME NOT NULL DEFAULT (datetime('now')),
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (spot_id) REFERENCES fishing_spots(id)
);
CREATE INDEX idx_checkins_user_spot ON spot_checkins(user_id, spot_id);
CREATE INDEX idx_checkins_time ON spot_checkins(checkin_time);
```

### 2.2 新增表：share_tokens（分享Token）

> **说明：** 微信定向分享时生成加密Token，有效期内可访问，过期后自动失效并由定时任务清理。

| 字段 | 类型 | 约束 | 说明 |

## 4. 规则引擎设计（AI推荐）

### 4.1 评分体系

**总分 = 气压分 × 0.35 + 水温分 × 0.25 + 风力分 × 0.20 + 月相分 × 0.10 + 时段分 × 0.10**

各维度分值范围 0-100，100为最优。

## 5. 天文计算

### 5.1 月相计算

采用"已知新月基准日"算法：月球朔望周期 ≈ 29.53058867 天，以已知新月日期为基准计算目标日期月相。

**基准：** 2000年1月6日 18:14 UTC 为新月（朔）。

```python
# app/services/astronomy_service.py

from datetime import datetime, timezone
import math

# 已知新月基准：2000-01-06 18:14 UTC
KNOWN_NEW_MOON = datetime(2000, 1, 6, 18, 14, 0, tzinfo=timezone.utc)
SYNODIC_MONTH = 29.53058867  # 朔望月天数

MOON_PHASES = [
    (0.00, "新月", "new_moon"),
    (0.125, "蛾眉月", "waxing_crescent"),
    (0.25, "上弦月", "first_quarter"),
    (0.375, "盈凸月", "waxing_gibbous"),
    (0.50, "满月", "full_moon"),
    (0.625, "亏凸月", "waning_gibbous"),
    (0.75, "下弦月", "last_quarter"),
    (0.875, "残月", "waning_crescent"),
]


def calculate_moon_phase(dt: datetime) -> tuple[str, str, float]:
    """
    计算指定日期的月相。
    Returns: (月相名称, 月相代码, 月球照明度 0.0-1.0)
    """
    if dt.tzinfo is None:
        dt = dt.replace(tzinfo=timezone.utc)
    else:
        dt = dt.astimezone(timezone.utc)

    delta = (dt - KNOWN_NEW_MOON).total_seconds() / 86400
    phase = (delta % SYNODIC_MONTH) / SYNODIC_MONTH  # 0.0-1.0
    illumination = (1 - math.cos(phase * 2 * math.pi)) / 2  # 照明度

    for threshold, name, code in reversed(MOON_PHASES):
        if phase >= threshold:
            return name, code, round(illumination, 2)
    return MOON_PHASES[0][1], MOON_PHASES[0][2], 0.0
```

### 5.2 日出日落计算

采用 Jean Meeus 天文公式，输入纬度/经度/日期，输出日出/日落/晨光始/昏影终时刻。

```python
# app/services/astronomy_service.py

def calculate_sun_times(lat: float, lng: float, dt: datetime) -> dict[str, str]:
    """
    计算指定地点和日期的日出日落时刻。
    返回: {sunrise, sunset, dawn, dusk} 格式 "HH:MM"
    太阳在地平线下18度为晨昏影边界。
    """
    import math

    year, month, day = dt.year, dt.month, dt.day

    # 儒略日
    if month <= 2:
        year -= 1
        month += 12
    A = int(year / 100)
    B = 2 - A + int(A / 4)
    JD = int(365.25 * (year + 4716)) + int(30.6001 * (month + 1)) + day + B - 1524.5

    n = JD - 2451545.0
    L = (280.46 + 0.9856474 * n) % 360
    g = (357.528 + 0.9856003 * n) % 360
    eclon = L + 1.915 * math.sin(math.radians(g)) + 0.020 * math.sin(math.radians(2 * g))
    obliquity = 23.439 - 0.0000004 * n

    RA = math.degrees(math.atan2(
        math.cos(math.radians(obliquity)) * math.sin(math.radians(eclon)),
        math.cos(math.radians(eclon))
    ))
    dec = math.degrees(math.asin(
        math.sin(math.radians(obliquity)) * math.sin(math.radians(eclon))
    ))

    lat_rad = math.radians(lat)
    dec_rad = math.radians(dec)

    def hour_angle(alt: float) -> float:
        cos_H = (math.sin(math.radians(alt)) - math.sin(lat_rad) * math.sin(dec_rad)) \
                / (math.cos(lat_rad) * math.cos(dec_rad))
        if cos_H > 1 or cos_H < -1:
            return float('nan')
        return math.degrees(math.acos(cos_H))

    H_sun = hour_angle(0)    # 日出日落
    H_dawn = hour_angle(-18) # 晨光始/昏影终

    noon = 12 - (lng / 15) - 0.0053 * math.sin(math.radians(g)) \
           + 1.0028 * math.sin(math.radians(2 * (L - lng)))

    def fmt(h: float) -> str:
        if math.isnan(h):
            return "--:--"
        h = max(0, min(24, h))
        return f"{int(h):02d}:{int((h - int(h)) * 60):02d}"

    return {
        "sunrise": fmt(noon - H_sun / 15),
        "sunset":  fmt(noon + H_sun / 15),
        "dawn":    fmt(noon - H_dawn / 15),
        "dusk":    fmt(noon + H_dawn / 15),
    }
```

## 9. 定时任务（可选）

### 清理过期分享Token

初期采用「惰性删除」：访问 token 时检查是否过期，过期则返回 410 并删除。

如需主动清理，可增加后台定时任务（APScheduler 每日凌晨执行）：

```python
async def cleanup_expired_tokens():
    from sqlalchemy import delete
    from app.database import async_session_maker
    from app.models.share_token import ShareToken
    from datetime import datetime, timezone

    async with async_session_maker() as db:
        stmt = delete(ShareToken).where(
            ShareToken.expires_at < datetime.now(timezone.utc)
        )
        result = await db.execute(stmt)
        await db.commit()
        return result.rowcount
```
