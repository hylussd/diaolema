# 钓了吗 P0 MVP 目录结构

## 1. 整体结构

```
钓了吗/
├── miniapp/                 # 微信小程序前端
```

### 关键文件说明

| 文件 | 职责 |
|------|------|
| `app/main.py` | FastAPI 实例、跨域、CORS、中间件注册 |
| `app/config.py` | 环境变量（和风KEY、数据库路径、微信配置） |
| `app/database.py` | aiosqlite 连接池、session 管理 |
| `app/models/*.py` | SQLAlchemy ORM 模型定义 |
| `app/schemas/*.py` | Pydantic 序列化/反序列化模型 |
| `app/routers/*.py` | API 路由定义（@router.get/post/...） |
| `app/services/` | 业务逻辑（天气缓存、地理计算、小程序码） |
| `app/dependencies.py` | `get_current_user`（从 X-Openid 解析用户） |
| `scripts/init_db.py` | 启动时建表（CREATE TABLE IF NOT EXISTS） |

---

## 4. 文件编码约定

- 前端：小程序**不支持**中文路径，目录名全英文
- 后端：Python 源码 UTF-8，无中文变量名，注释可用中文
- 数据库：SQLite 默认 UTF-8

---

## 5. 环境变量清单

后端 `.env` 需配置：

```
# 数据库
DATABASE_URL=sqlite+aiosqlite:///./data/diaolema.db

# 和风天气
QWEATHER_API_KEY=your_hefeng_key
QWEATHER_LOCATION_ID=   # 城市 location ID（可选，用 lat/lng 查询）

# 微信小程序
WECHAT_APPID=wx_your_appid
WECHAT_SECRET=your_secret
WECHAT_MASTER_KEY=your_master_key   # 生成小程序码用

# 服务
HOST=0.0.0.0
PORT=8000
DEBUG=true
CORS_ORIGINS=["http://127.0.0.1:8080"]
```
